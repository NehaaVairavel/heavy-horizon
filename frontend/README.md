# Heavy Horizon

Construction Equipment Rental & Sales application built with React, Vite, and Tailwind CSS.

## Project Info

Premium backhoe loaders and excavators for rent and sale.

## Getting Started

To work locally on this project:

1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
```

2. Install dependencies
```sh
npm install
```

3. Start the development server
```sh
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
